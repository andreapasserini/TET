To run the bibliometrics classification experiment on the AMiner data set, please use the following command:

bash run_biblio_classification.sh localhost/imdb YOUR_MYSQL_USER YOUR_MYSQL_PASSWORD

or change the run_biblio_classification.sh accordingly.

* The TET used in this experiment is in the tet-apc file. It is the author-paper-citation single branch TET.

* The training and test data sets are in the data.train and data.test files, with corresponding labels.train and labels.test files.
